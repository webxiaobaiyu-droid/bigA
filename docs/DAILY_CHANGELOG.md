# BigA 每日开发更新记录

本文按日期记录 BigA 已完成的产品功能、数据接入、交互优化与工程化工作。详细 API 字段和代码结构见 [项目逻辑与 API 说明](PROJECT_ARCHITECTURE_AND_APIS.md)。

## 2026-07-12

主题：行情可信度、交易日历、请求稳定性、K 线复权与市场复盘。

### 第三阶段：市场复盘与可观测性

- 市场情绪增加 `实时 / 复盘` 模式，每 5 分钟在本机保存一个紧凑快照。
- 使用 ECharts 展示情绪分、红盘覆盖和封板率轨迹；点击采集点可查看当时涨跌停、炸板、连板高度和指数均值。
- 资金流向增加 `实时榜 / 轮动` 模式，使用热图展示行业净流入占比随时间的变化。
- 行业轮动增加连续流入/流出交易日、最近五个快照净额、资金加速度、排名变化和轮动状态。
- 同一交易日多个盘中快照只用于变化分析，不会被误计为多个连续交易日。
- 顶部“数据源”按钮接入运行质量面板，展示各主机请求量、成功率、平均/最近延迟、重试和熔断状态。
- 新增 `pnpm run validate:market-history`，覆盖 5 分钟桶去重、交易日连续性、加速度和轮动状态。
- 数据源健康增加 `现场 / 趋势` 视图：按上海日期保存每个主机的请求质量，跨启动保留最近 90 天。
- 趋势页使用 ECharts 对比指定主机每日成功率和平均延迟，并提供日聚合明细。
- 新增 `pnpm run validate:transport`，覆盖主机日聚合的成功、失败、重试与延迟计算。

### 外观与主题

- 增加深色和浅色两种颜色模式，浅色使用独立的背景、文字、边框和图表对比度体系。
- 增加金融金、量化蓝、墨玉青、朱砂红四套终端主题，主题色与涨跌语义色相互独立。
- 终端设置接入外观弹窗，支持即时预览、恢复默认和本地持久化；启动前应用已保存配置，避免主题闪烁。
- 将全局页面、表格、卡片、弹窗和控件迁移为 CSS 语义变量。
- ECharts 与 `lightweight-charts` 支持运行时重绘，切换模式或主题后同步更新网格、坐标轴、Tooltip 和关键线条。
- 新增 `pnpm run validate:appearance`，覆盖默认配置、四套主题、浅色变量和 CSS 循环引用检查。

### 交易时段与数据状态

- 增加 Asia/Shanghai 时区下的 A 股交易日历，识别周末和已配置的 2026 年休市日。
- 精确区分盘前、开盘集合竞价、连续竞价、午间休盘、收盘集合竞价和盘后，并给出下一状态时间。
- 对未配置节假日表的年份标记为“仅工作日推断”，避免把推断结果包装成官方交易日历。
- 东方财富报价使用 `f124` 供应商时间，新浪报价使用接口日期与时间字段，日线兜底使用 K 线自身日期。
- 去掉缺失供应商时间时的本机当前时间伪装，缺失时沿用上一条真实时间或显示无有效更新时间。
- 热门板块和热门股票拆分“供应商行情时间”与“BigA 拉取完成时间”：东方财富使用 `f124`，新浪未提供榜单时间时明确显示“拉取”，不再显示为实时行情。
- 知识库从历史 K 线生成的案例使用最后一根 K 的交易日期，避免将本地扫描时间写成股票更新时间。
- 顶部行情状态增加实时、延迟、陈旧、回退、异常、待数据和休市七种健康状态；休市时不会把上一交易日收盘误报为陈旧行情。

### 请求稳定性

- 相同 URL 的并发请求复用同一个 Promise，减少页面切换和多组件加载造成的重复请求。
- 行情请求增加 12 秒超时、一次短延迟重试，以及连续失败后的 15 秒主机级熔断。
- 开发代理失败时直接暴露失败，不再静默改走浏览器直连而掩盖跨域或代理问题。
- 热门板块与热门股票列表增加请求序号和筛选快照，快速切换 Tab 或排序时丢弃过期响应。

### K 线复权与校验

- 日、周、月 K 增加前复权、不复权、后复权切换，自选详情与热门股票详情均可使用。
- 复权偏好写入 `biga.adjustment.v1`，不同股票、周期和复权方式使用独立内存缓存。
- 东方财富 K 线按 `fqt=0/1/2` 请求不复权、前复权和后复权；分时不参与复权。
- 东方财富历史接口不可用时，不复权数据回退新浪日 K；前/后复权使用新浪真实 `qfq/hfq` 因子调整原始日 K，不使用未调整价格冒充复权结果。
- 增加 `pnpm run validate:core`，覆盖交易时段边界、节假日、行情健康状态和复权映射。

### 自选分组与批量管理

- 自选股增加 `短线 / 趋势 / 观察 / 持仓` 默认分组，并支持创建、重命名和删除自定义分组。
- 分组与股票元数据分离保存到 `biga.watchlist-organization.v2`，旧自选数据自动迁移到观察组。
- 支持为股票添加最多 6 个本地标签，标签直接显示在自选列表。
- 增加自定义排序和原生拖动调整顺序；切换分组后只展示该组股票。
- 增加自选管理弹层，支持逐只改分组/标签、复选和批量删除。
- 支持 BigA JSON 与标准 CSV 导入导出；CSV 解析兼容引号、逗号和中文字段。

### 提醒闭环

- 建立纯函数提醒引擎，支持价格上下破、涨跌幅、量比、成交额、换手率、日线放量、MA5/MA10 交叉、涨停开板和板块异动 12 类规则。
- 提醒使用条件上升沿触发，并同时执行规则冷却和单股票系统通知频率限制，避免持续命中反复通知。
- 处理 ST、主板、创业板、科创板、北交所不同涨跌停比例；N/C 新股不执行开板提醒。
- 数据缺失时保持原条件状态，不将接口短暂空值视为条件解除并重新触发。
- 增加提醒中心的规则、记录、设置三视图，支持启停、删除、已读、全部已读和点击记录回到股票。
- 自选列表显示最近触发规则和持续时间，股票右侧栏展示启用规则数与最近触发时间。
- Electron 接入 macOS 系统通知；点击通知聚焦窗口并返回对应自选股。
- 支持跨午夜静默时段、全局启停、系统通知开关和默认冷却时间。
- 增加 `pnpm run validate:alerts`，覆盖边沿、冷却、缺失数据、指标、均线、量能、板块与涨跌停边界。

### 第二阶段收尾

- 自选分组支持颜色修改、上下排序与删除前确认；重名或空名称不会覆盖已有分组。
- 自选导入改为显示新增、更新与跳过行数；错误格式和没有有效股票时明确提示失败原因。
- 提醒规则支持创建后的重新编辑，可修改类型、阈值、冷却时间及作用范围。
- 提醒范围支持全部自选、多个指定股票、动态自选分组和所属板块；分组规则会跟随分组成员变化实时生效。
- 提醒历史增加股票/规则关键词筛选、未读筛选、CSV 导出和清空操作。
- 新增 `pnpm run validate:watchlist`，覆盖 CSV/JSON 导入报告、分组更新排序、元数据与 CSV 转义。

## 2026-07-11

主题：市场监控深化、知识库训练系统、K 线完善、F10 可视化与桌面打包。

### 市场监控

- 将市场监控明确拆分为 `市场情绪`、`龙虎榜`、`资金流向` 三个独立视图。
- 接入全市场上涨、平盘、下跌家数，以及涨停、跌停、炸板、封板率和连板高度。
- 接入上证指数、深证成指、创业板指实时快照。
- 建立可解释的市场情绪分，按红盘覆盖、封板率、涨跌停平衡和指数表现计算冰点至过热区间。
- 使用 ECharts 重做市场情绪驾驶舱：
  - 市场温度仪表盘。
  - 上涨、平盘、下跌分布环图。
  - 涨停、炸板、跌停质量柱状图。
  - 连板核心横向排行。
  - 红盘广度、涨停优势、封板质量、连板高度、指数动能五维结构雷达。
- 保留可点击的涨停个股列表，用于从市场结构下钻至具体股票行情。

### 龙虎榜与席位

- 接入东方财富龙虎榜日榜、买方席位和卖方席位数据。
- 增加龙虎榜本地筛选，可按股票名称、代码、席位和上榜原因检索。
- 龙虎榜个股详情支持日 K 与分时切换。
- 日 K 展示机构、沪深股通和已识别游资的公开 B/S 标记。
- 分时展示基于逐笔主动买卖方向推演的 `B* / S*`，并明确标识其不是席位精确成交时点。
- 右侧席位支持按机构、知名游资和营业部过滤。
- 点击具体席位后，只展示该席位对应的日 K B/S 和分时推演标记。
- 同交易日、同席位代码或规范化席位名称的买卖记录合并为一行，买入和卖出累加后重算净额。
- 席位列表支持按买入、卖出、净额升降序排列。
- 固定席位名称列宽，超长名称使用省略号并通过 Hover 展示全文。

### 知识库与训练

- 知识库升级为独立顶部工作区，不与自选股或其他侧栏耦合。
- 内置 52 个面向入门和进阶投资者的交易术语，覆盖量价、盘口、情绪、资金、风控和市场黑话。
- 增加词云、分类筛选、关键词搜索、概念解析和风险提示。
- 建立六类真实行情模式扫描：量价齐升、放量突破、缩量回踩、冲高回落、量价背离和均线多头。
- 每个规则展示命中证据、阈值、后续 1/3/5/10 日表现和失败原因。
- 增加正例、反例、临界案例和真实股票教学标注。
- 增加规则实验室，可本地调整阈值并重新扫描，不伪装成服务端策略结果。
- 增加未来数据遮挡回放、判断训练、收藏和本地错题本。
- 教学图支持辅助线、文字解释、成交量、MACD、RSI 和 KDJ。

### 分时与 K 线

- 修复午间休市后的时间轴连接，早盘与午盘按 A 股完整交易时段展示。
- 分时横轴统一为 15 分钟刻度，`11:30 / 13:00` 使用合并边界标签。
- 补齐截止当前时间之后的空白交易时段，确保全天时间轴固定。
- 修复下午成交量因压缩时间戳不一致而错位或缺失的问题。
- 分时价格坐标以当日涨停价和跌停价为上下限，并显示昨收 `0%` 参考线。
- 日、周、月 K 线底部显示完整年份和日期。
- 日、周、月 K 支持悬停速览和单击锁定周期详情。
- 周期详情展示开高低收、涨跌幅、振幅、成交量、成交额、换手率、均线、上下影线和副图指标。
- 支持左右方向键切换周期，`Esc` 取消锁定。
- 副图支持 `VOL`、`MACD`、`RSI`、`KDJ`，选择结果本地持久化。
- 禁止滚轮缩放、鼠标拖动、触摸拖动、捏合缩放、坐标轴拖动和惯性滚动，避免监盘误操作。

### 个股资料与财务

- 个股详情下方增加 `资讯`、`财务`、`公司`、`公告` 四个 F10 视图。
- 接入个股新闻、主要财务指标、公司资料和公告数据。
- 财务历史报告由 8 期扩展到 20 期。
- 使用 ECharts 展示营收、归母净利润、ROE、毛利率、负债率、EPS 和每股经营现金流。
- 营收和归母净利润支持单季度、累计和年度口径。
- 单季度值由同年累计报告差分得到，并重新计算同季度同比，避免把累计值误标为单季值。
- 财务图支持双轴柱线对比、Tooltip、柱顶数值、时间缩放和指标切换。
- ECharts 财务组件按需异步加载，不增加主监盘首屏负担。

### 新闻

- 修复东方财富列表地址指向空正文页面的问题，优先使用规范的 `/a/文章ID.html` 地址。
- 东方财富正文解析兼容多种文章容器和 JSON-LD 回退。
- 接入网易财经文章完整正文，并补充 Electron 新闻域名白名单。
- 去掉正文最多 80 段的人为限制；快讯仍按接口返回的完整电报原文展示。
- 新闻详情改为应用内二级页面，不再调用外部网页。
- 正文使用居中的阅读列：标题区约 840px，正文区约 760px，并优化来源、标签、段落间距和行高。

### 自选股与跨页面联动

- 股票详情标题支持直接移出自选。
- 热门股票详情支持 `加自选 / 删自选`。
- 龙虎榜席位详情支持 `加自选 / 删自选`。
- 板块成分股、情绪榜和资金榜进入股票详情后可继续操作自选。
- 所有入口共用 `watchMetas + MarketDataProvider + localStorage`，状态即时同步。
- 添加股票时自动识别沪市、深市和北交所，并立即启动行情订阅。

### 工程化与桌面端

- 增加 macOS arm64 DMG 打包配置、应用图标和本地无签名构建流程。
- 生成并安装首个生产版 `/Applications/BigA.app`。
- 将 ECharts 财务与情绪图表拆为异步组件，保持主包和首屏加载轻量。
- 完成项目架构、数据流、API 字段、刷新策略、回退逻辑与数据可信边界文档。
