把看盘信息压到一屏里。

Workspace

不是资讯堆叠，而是操盘顺序。

先判断市场情绪，再看板块资金，之后进入个股分时和 K 线。BigA 的页面按这个顺序组织，减少反复跳转，也给后续策略和自选股计划留接口。

01

分时 / K 线

涨跌停边界、0% 中轴、午盘间隔、成交量和 MACD 指标联动。

02

龙虎榜 B/S

席位合并、机构过滤、买卖点映射到分时，方便看清资金行为。

03

新闻与财务

东方财富、网易财经与企业财务信息统一整理，股票详情内直接查看。

04

知识库案例

量价齐升、日内做 T、小作文等知识点结合真实 K 线做教学标注。

Data Pipeline

真实数据优先，本地能力兜底。

行情以东方财富为主，新浪与腾讯补充盘口、资金与排行。新闻和公告会标准化成本地模型；市场情绪、行业资金、知识库案例和预警在本机侧计算。

biga.config
market.provider = eastmoney

fallback.quote = sina
fallback.rank = tencent

news.primary = eastmoney
news.backup = netease

signals.local = true
alerts.local = true
