KEEP THE CONTEXT

不是再放一个行情软件。
而是整理看盘时的顺序。

开盘后先看市场能否承接，再确认资金在哪些方向聚集，最后才进入个股。BigA 的六个工作区按这个节奏组织，关键信息不会藏在另一个窗口。
