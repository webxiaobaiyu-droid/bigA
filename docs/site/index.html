Monitor Workspace

面向操盘视角的桌面工作区

分时

价格、成交量和指标联动

支持 A 股午休时段坐标、涨跌停边界、0% 中轴、MACD 等常用指标，减少看盘时的来回切换。

资金

板块监控与资金流向

将市场情绪、行业资金、热门板块、热门个股拆成独立顶部标签，便于快速判断当日主线。

龙虎榜

机构席位 B/S 点位

龙虎榜支持本地过滤、席位合并、排序，以及点击机构查看对应分时买卖点。

知识库

从术语到案例复盘

内置量价、做 T、消息面和小作文等知识点，并结合真实 K 线案例输出标注、辅助线和解析。

Data Sources

行情与 K 线

东方财富为主，新浪和腾讯作为部分盘口、资金、排行与回退数据源。

新闻与公告
